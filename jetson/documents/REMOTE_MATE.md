# Jetson NANOへのリモート接続(ubuntu-mate-desktopを使用)

## 動作環境

- ホストPC
  - ASUS UX450FDX
  - Windows10 Home
    - Tera Term 4.104
  - Ubuntu 18.04 LTS
    - Linux Kernel 4.15.0-66-generic

- Jetson NANO
  - JetPack 4.2.1
    - Ubuntu 18.04 LTS
    - Linux Kernal 4.9.140-tegra
    - CUDA toolkit 10
  - SANDISK microSD CLASS 10 UHS-1 U3 64GB
